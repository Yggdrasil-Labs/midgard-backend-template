# Quality Score

本文件是 Midgard 文档与工程骨架的质量评分卡。

目的不是追求绝对分数，而是持续暴露最影响协作效率的缺口。

## 评分标准

每项满分 5 分：

- 1: 基本缺失
- 3: 可用但不稳定
- 5: 清晰、可维护、可验证

## 当前最值得提升的三项

1. 为文档地图增加最小一致性检查。
2. 为数据库与示例聚合提供更真实的生成型参考。
3. 将可靠性和安全约束逐步对应到代码或配置样板。
